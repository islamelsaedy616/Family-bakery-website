# Family Bakery — Responsive Landing Page

A modern and responsive bakery landing page built using pure HTML and CSS.  
Designed to highlight bakery products, share the story of the brand, display the shop’s location, and provide contact details — all in a clean and elegant layout.

## Features

- Fully responsive design optimized for desktop and tablet screens.
- Well-structured sections: Home, About Us, Our Location, Our Baking, Contact Information, and Footer.
- Integrated Google Maps for easy location display.
- Clean, semantic HTML and organized CSS for better readability and maintenance.
- Easy customization for text, images, colors, and overall style.
- Perfect for portfolio presentation or as a base template for small business websites.
